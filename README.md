## Data Science

### [Video Game Sales Analysis](./dataScience/finalProject)
EDA on 16k game titles (1980–2016). Four questions investigated with statistical rigor:
- Genre bias by platform → bootstrap hypothesis test (99% CI)
- Genre popularity variance over time → sales fraction by release year
- Regional genre bias → permutation test (99.9% significance)
- Predicting Japan-majority sales → Gradient Boosting classifier (F1: 0.74, accuracy: ~0.91)
